Data Collection

- We will not collect any data about you/your server if you're not used any commands or added Studio to your server. - Stored data is not shared to anyone. Data can only be accessed by Studio and it’s Owners. - Properties that are given down below are collected and stored data in database if needed. - Guild ID (Server ID) - User ID - Role ID - Channel ID - Message are temporally stored and deleted after 60-120 seconds. This time interval depends on the specific command in use.. This cache process required to text protections perform better and faster.


Data Deletion

- If you kick the bot from your server, your data will be deleted. - We are not storing any kind of message data permanently. All message data will be deleted eventually. - You can personally request for user specific data to be deleted via the discord support server. (https://discord.gg/Bz7nArkZEx)
